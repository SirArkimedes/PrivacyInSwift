## Requirements

```ruby
>= iOS 10.0
``` 

## Installation

PrivacyInSwift is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'PrivacyInSwift'
```
